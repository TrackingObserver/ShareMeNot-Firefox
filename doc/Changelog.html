ShareMeNot for Firefox

Version 2.3.1

  • Reduced size of the LinkedIn replacement button.

Version 2.3

Released on 2013-06-19 .

  • Fixed issue with add-on bar icon, and repacked with SDK for Firefox 22+.

Version 2.2

Released on 2012-12-30 .

  • Fixed issue that caused problems on Twitter itself.

Version 2.1

Released on 2012-11-11 .

  • Added an option to remove cookies from requests to trackers and load the real social buttons, rather than replacing the buttons entirely. This option is less privacy-preserving, but will not cause page layout issues.

Version 2.0.2

Released on 2012-07-08 .

  • Reduced the size of the Facebook replacement button and the +1 replacement button to prevent ShareMeNot from breaking page layouts.
  • Fixed an issue in which the content script was throwing exceptions that were logged to the system log.

Version 2.0.1

Released on 2012-07-04 .

  • Removed debug logging that was accidentally included in the 2.0 release.

Version 2.0

Released on 2012-06-29 .

  • Completely rewritten from scratch using the Firefox Add-On SDK.
  • Shares the same code base as ShareMeNot for Chrome 1.0.
  • Blocks entire web requests to trackers (like ShareMeNot for Chrome) instead of just cookies. This enhances privacy by preventing browser fingerprinting.
  • Replaces the blocked tracker buttons with images stored on your computer.
  • Can choose which trackers to block and unblock through the ShareMeNot icon in the add-on bar.
  • No longer requires restarting Firefox to install.
  • Settings are now built into the Firefox add-on management page instead of being in a separate window.
